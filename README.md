## Key Project

### LingMarket (Wearable Dictionary Engine)

- Offline search over large-scale word corpus (~50k entries)  
- Optimized for low-end wearable hardware  
- Custom indexing strategy to reduce IO and CPU spikes  
- Dual input system (custom keyboard + system keyboard)
